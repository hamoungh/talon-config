### Talon voice cheatsheet — Windows

#### Editing

| Spoken | Key | Spoken | Key |
|--------|-----|--------|-----|
| copy that | `Ctrl+C` | cut that | `Ctrl+X` |
| pace / paste that | `Ctrl+V` | select all | `Ctrl+A` |
| undo that | `Ctrl+Z` | redo that | `Ctrl+Y` |
| file save | `Ctrl+S` | file save all | `Ctrl+Shift+S` |

#### Navigation (via edit actions)

| Spoken | Key | Spoken | Key |
|--------|-----|--------|-----|
| word left | `Ctrl+←` | word right | `Ctrl+→` |
| extend word left | `Ctrl+Shift+←` | extend word right | `Ctrl+Shift+→` |
| line start | `Home` | line end | `End` |
| extend line start | `Shift+Home` | extend line end | `Shift+End` |
| file start | `Ctrl+Home` | file end | `Ctrl+End` |
| extend file start | `Ctrl+Shift+Home` | extend file end | `Ctrl+Shift+End` |
| page up | `PageUp` | page down | `PageDown` |
| find | `Ctrl+F` | find next | `F3` |
| find previous | `Shift+F3` | print | `Ctrl+P` |
| zoom in | `Ctrl++` | zoom out | `Ctrl+-` |
| zoom reset | `Ctrl+0` | | |

#### Tabs (active in any tabbed app)

| Spoken | Key | Spoken | Key |
|--------|-----|--------|-----|
| tab open | `Ctrl+T` | tab close | `Ctrl+W` |
| tab next | `Ctrl+Tab` | tab previous | `Ctrl+Shift+Tab` |
| tab reopen | `Ctrl+Shift+T` | window new | `Ctrl+N` |

#### Window management

| Spoken | Action | Spoken | Action |
|--------|--------|--------|--------|
| window close | `Alt+F4` | window hide | `Alt+Space N` |
| focus last | `Alt+Tab` | focus `<app>` | Focus named app |
| focus | App switcher menu | running list | Show/hide running apps |
| launch `<app>` | Launch app | | |

#### Window snapping — `snap <position>`

**Halves / Thirds**

| Spoken | Spoken | Spoken | Spoken |
|--------|--------|--------|--------|
| `left` | `right` | `top` | `bottom` |
| `left third` | `center third` | `right third` | |
| `left two thirds` | `right two thirds` | | |

**Quarters**

| Spoken | Spoken | Spoken | Spoken |
|--------|--------|--------|--------|
| `top left` | `top right` | `bottom left` | `bottom right` |

**Sized variants (small / large)**

| Spoken | Spoken | Spoken | Spoken |
|--------|--------|--------|--------|
| `left small` | `left large` | `right small` | `right large` |
| `center small` | `top left small` | `top right small` | `bottom left small` |
| `bottom right small` | `top left large` | `top right large` | |

**Multi-screen**

| Spoken | Action |
|--------|--------|
| `snap next screen` | Move window to next screen |
| `snap last screen` | Move window to previous screen |
| `snap screen <N>` | Move window to screen N |
| `snap <app> <position>` | Focus app and snap it |
| `full` / `fullscreen` / `center` | Full / fullscreen / centred |
